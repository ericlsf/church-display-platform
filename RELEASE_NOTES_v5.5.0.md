# v5.5.0 — Actionable Health Diagnostics

## Added

Every display health score now includes its five contributing checks:

- Hub connection
- Display application
- Content-folder assignment
- Local media availability
- Last sync result

Each failed check shows:

- why the health score was reduced;
- what the operator should check;
- a direct corrective action.

## Available actions

- Restart Player
- Retry Sync
- Assign Folder
- Reboot Device

Health Diagnostics appears above Recent Jobs on the individual display page.
