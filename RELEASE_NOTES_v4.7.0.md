# v4.7.0 — Scheduled and staged rollouts

## Added

- Schedule display-software deployments for a future date and time.
- Choose one canary display before updating the rest.
- Automatically pause when the canary fails.
- Optionally pause after any later display failure.
- Resume or cancel a rollout from the Hub.
- Live rollout timeline with per-display progress.
- Dry-run rollout support.
- Background rollout processor running every 30 seconds.

## Workflow

1. Open Scheduled Rollouts.
2. Choose a version and displays.
3. Optionally choose a canary.
4. Select a start time.
5. Schedule the rollout.
6. Monitor the timeline and resume or cancel if needed.
