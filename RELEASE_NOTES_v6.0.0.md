# v6.0.0 — Display Details UX Refresh

## Operator header

The display page now begins with:

- display name;
- online/offline status;
- heartbeat age;
- installed version;
- sync state;
- direct links to Content, Health, and Software.

## Summary row

Four scan-friendly cards show:

- Health
- Media
- Content Folder
- Software

## Device Health

Health is now a compact checklist with:

- clear passing/failing state;
- short failure explanation;
- direct corrective action.

## Layout

Spacing, section padding, helper text, and mobile behavior are tightened so the
page is faster to scan without removing deeper operational sections.
