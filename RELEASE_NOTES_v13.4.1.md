# v13.4.1 — Fleet Overview integration correction

This corrective release combines the v13.4.0 Fleet Overview with the Hub features
that shipped locally in v13.1.0 through v13.3.1 but were not present on the remote
`main` branch when v13.4.0 was created.

## Included

- v13.4 responsive Fleet Overview cards, filters, health states, quick actions,
  bulk operations, live activity, and notifications.
- v13.1 deployment center and fleet operations.
- v13.1.1 telemetry compatibility improvements.
- v13.2 secure remote-access workspace.
- v13.3 remote screenshot workspace.
- v13.3.1 live device health fixes and polish.
- Full Command Center and shared application-shell compatibility.

No configuration migration is required.
