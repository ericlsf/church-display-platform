# v5.2.0 — Visual Polish and Live Health

## Added

- Consistent spacing, radii, controls, and action styling.
- Improved responsive layouts for tablets and phones.
- Stronger status pills and empty states.
- Live device-health refresh every 15 seconds.
- Screenshot preview when supplied by the display heartbeat.
- CPU temperature, storage usage, media count, version, and heartbeat metrics.
- Better card alignment across displays, profiles, fleet map, and operations pages.

## Compatibility

The health panel only displays metrics already reported by a display. Missing
metrics show as `Unknown`; the page remains functional.
