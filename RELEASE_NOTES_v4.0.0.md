# v4.0.0 — Production UX release

## Changed

- Replaced expanding dropdown navigation with five workflow-based sections.
- Added contextual section navigation for Content, Fleet, Operations, and Administration.
- Redesigned Home around system health, needs-attention items, quick actions, live work, publishing, and display snapshots.
- Added a keyboard command palette (`Ctrl+K` / `Cmd+K`) for fast navigation and global search.
- Reduced default dashboard density and moved secondary activity into expandable details.

## Added

- One-command production readiness check.
- One-to-two-day production go-live and rollback checklist.

## Scope

This release intentionally avoids database, API, content-sync, and deployment-protocol changes. It is suitable for a short production-cut window after normal validation and one-display staging.
