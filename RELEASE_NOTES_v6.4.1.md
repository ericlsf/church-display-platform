# v6.4.1 — Fleet Operations Form Layout

## Changed

Fleet Operations settings are no longer compressed into one horizontal row.

Each value now appears on its own line:

- Bulk Action
- Version
- Mode
- Overlay Enabled
- Overlay Text
- Clock Enabled
- Countdown Enabled
- Countdown Window
- Image Duration
- Override Maintenance Protection

Checkboxes remain compact and left aligned. Text fields and dropdowns use the
available card width, while action buttons remain grouped beneath the settings.
