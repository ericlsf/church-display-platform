Secure setup

  1. Create a Cloudflare Zero Trust self-hosted application and an Allow policy for approved accounts.
  2. Create a remotely managed Tunnel with a public hostname pointing to http://localhost:8090.
  3. Copy only the eyJ… tunnel token, then run the helper command on the Hub.
sudo /home/lsfservice/church-display-platform/scripts/configure-remote-access.sh \
  --token 'PASTE_TUNNEL_TOKEN' \
  --hostname 'hub.example.org'

Important: create the Cloudflare Access application before enabling the public hostname. The Hub also retains its own login, providing a second authentication layer.
