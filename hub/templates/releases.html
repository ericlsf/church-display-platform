Build a Release Package

Run this from the repository root after committing and tagging:

./release/build_release.py v1.8.0

The release builder creates:

  • release/dist/church-display-platform-vX.Y.Z.zip
  • release/dist/manifest.json
  • release/dist/SHA256SUMS
  • release/dist/release_notes.md

Verify a package with:

./release/verify.py release/dist/church-display-platform-v1.8.0.zip --manifest release/dist/manifest.json
