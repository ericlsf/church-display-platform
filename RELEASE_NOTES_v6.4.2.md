# v6.4.2 — Fleet Operations Control Groups

The Fleet Operations form now uses these rows:

1. Bulk Action
2. Version | Mode
3. Overlay Enabled | Overlay Text
4. Clock Enabled
5. Countdown Enabled
6. Countdown Window
7. Image Duration
8. Maintenance Override | Queue Button

The grouping script identifies controls by their labels, preserving the
existing form fields and submission behavior while improving visual structure.
