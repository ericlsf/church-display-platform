# Church Display Platform v11.1.0

## Display workspace redesign

- Replaces free-text sync settings with a Google Drive-only content assignment workflow.
- Uses cached Google Drive folders in real dropdowns.
- Keeps profile assignment in a managed dropdown.
- Presents groups as compact selectable chips.
- Reduces display-card height with a side-by-side preview and assignment layout.
- Moves presentation, connection, and removal controls into an Advanced popover.
- Adds multi-display bulk assignment for folders, profiles, and group membership.
- Keeps all Google Drive discovery out of the page request path by using the cached media index.
