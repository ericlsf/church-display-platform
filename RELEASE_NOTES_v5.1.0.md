# v5.1.0 — Operator Experience

## Added

- Guided new-display setup wizard.
- Rich display overview cards.
- Consistent Manage / Finish Setup actions.
- Breadcrumb navigation.
- Unified action bar on each display page.
- Last 10 Minutes activity timeline.
- Automatic activity refresh every 30 seconds.
- Combined recent jobs and audit activity.

## Operator workflow

1. Open `/operator/displays`.
2. Select **Finish Setup** for a new display.
3. Choose its group, content folder, display profile, and overlay.
4. Finish setup to queue sync and restart.
5. Use **Manage** for ongoing device operations.

- Setup wizard uses a Drive-backed content-folder dropdown.
