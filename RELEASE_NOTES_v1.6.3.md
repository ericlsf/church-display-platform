# v1.6.3 - Media Manager Playlist Ordering

Adds playlist reordering to the Media Manager.

## Highlights

- Browse Google Drive folders from the Hub.
- Save a custom playlist order per Drive folder.
- Drag-and-drop or arrow-button media reordering.
- Send saved playlist order to displays during sync.
- Display player respects saved playlist order after sync.
- Fixes Jinja dictionary key access for media item rendering.

## Notes

Playlist ordering is stored in `hub/config/playlists.json` and should remain local runtime data.
