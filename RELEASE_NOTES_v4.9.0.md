# v4.9.0 — Display Groups and Maintenance Mode

## Added

- Create reusable named display groups.
- Edit group membership.
- Delete groups.
- Per-display maintenance mode.
- Maintenance reason and timestamp.
- Foundation for group-targeted bulk actions and rollouts.

## Maintenance behavior

Displays under maintenance can be shown separately in fleet health, have
expected alerts suppressed, and be excluded from deployments unless explicitly
allowed.
