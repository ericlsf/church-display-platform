# v7.0.0 — Navigation and Information Architecture Redesign

## New application shell

The Hub now uses:

- A collapsible left sidebar
- A compact fixed top bar
- Global search
- Command palette shortcut
- Notification bell
- Administrator menu
- Page breadcrumbs
- Responsive mobile navigation

## Navigation structure

### Dashboard
- Overview
- Fleet Map

### Fleet
- Displays
- Groups
- Profiles

### Media
- Content Library
- Media

### Operations
- Bulk Operations
- Fleet Operations
- Jobs
- Alert Center
- History

### Administration
- Users
- Releases
- System
- Alert Rules
- Settings

## Decluttering

- Removes Home from the new primary navigation.
- Replaces the wide Notifications link with a bell.
- Moves administrator actions into a user menu.
- Groups related destinations instead of presenting every page as a top-level tab.
- Hides the legacy header and navigation at runtime.
- Retains existing routes and page functionality.
