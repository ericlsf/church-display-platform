# v3.0.0 — Multi-site management and simplified navigation

## Added
- Site/campus management.
- Direct display and group membership per site.
- Site-wide sync, health checks, restarts, reboots, playlist assignment, and app deployment.
- Online, outdated, playlist, and membership summaries per site.

## Changed
The Hub navigation is simplified into Dashboard, Operations, Content, Schedules, Fleet, and Manage.
All previous pages remain available inside the grouped menus.

## Storage
Site membership is stored in `hub/config/sites.json` and excluded from Git.
