# v5.0.0 — Unified Display Details

## Added

A single management page for each display:

- health and readiness;
- software version;
- playlist and media count;
- group membership;
- folder and overlay editing;
- profile application;
- maintenance mode;
- Sync, Restart Player, and Reboot actions;
- recent jobs;
- recent audit activity.

## URL

```text
/display/<display-id>
```

Example:

```text
/display/welcome-center
```
