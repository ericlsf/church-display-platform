# v1.7.0 Content Workflow Release

Adds a unified Content workflow page for browsing Google Drive media, editing playlist order, previewing playback sequence, deploying playlists to displays, and scheduling future playlist deployments.

Highlights:
- New `/content` page
- Folder browsing with breadcrumb navigation
- Drag-and-drop playlist ordering
- Browser-side playlist preview order
- Deploy playlist to selected displays
- Schedule playlist deployments
- Content library metadata cards
- Estimated playlist runtime for images
