# v5.9.0 — Live Content and Overlay Editor

## Added

- Drive-backed content-folder selector.
- Clear overlay, clock, and countdown sections.
- Live on-page display preview.
- Unsaved-change indicator.
- Browser warning before leaving with unsaved changes.
- Separate `Save Settings` and `Apply & Sync Now` actions.
- Responsive two-column layout with sticky preview on larger screens.

## Goal

Common content and presentation changes can now be completed from one section
without switching between multiple Hub pages.
