# Church Display Platform v9.5.0

## Media Library and Playlist Builder

- Rebuilds the existing Media Manager as a focused content workspace.
- Adds grid/list views with persisted preference.
- Adds instant folder search and media-type filters.
- Adds a compact drag-and-drop playlist builder with live numbering.
- Adds display selection with Select all/Clear all.
- Adds a sticky, task-oriented deployment workflow.
- Preserves the existing Drive analysis, order-save, cache-sync, and job APIs.
- Includes a timestamped rollback backup and regression tests.
