# Church Display Platform v11.0.0

## Fleet Management Workspace

- Replaces the basic display list with operational fleet cards.
- Shows online state, version, heartbeat age, current media, sync state, and latest preview.
- Adds one-card management for sync remote, folder/playlist, presentation profile, and group membership.
- Saves assignments centrally and queues the required display jobs.
- Adds one-click sync, player restart, reboot, and local administration actions.
- Keeps advanced presentation and connection controls available in each card.
- Uses the cached Drive media index for folder choices so the page does not invoke rclone.
