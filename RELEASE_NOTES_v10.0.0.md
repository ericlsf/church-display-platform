# Church Display Platform v10.0.0 — Unified Navigation

## Highlights

- Introduces six clear primary navigation areas: Dashboard, Command Center, Media Library, Displays, Schedules, and Settings.
- Makes the Command Center visible in the sidebar and command palette.
- Groups operational tools under Command Center so alerts, jobs, fleet operations, and history are easier to find.
- Groups deployment and rollout tools under Schedules.
- Updates route highlighting, breadcrumbs, and page labels throughout the application shell.
- Preserves all existing routes and functionality; this release changes navigation and information architecture only.
