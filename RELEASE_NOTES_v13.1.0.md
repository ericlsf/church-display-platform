# Church Display Platform v13.1.0

## Fleet Operations Foundation

- Adds a dedicated Operations navigation section.
- Adds a Deployment Center dashboard for content, software, and bulk delivery work.
- Moves deployment tools out of Schedules.
- Adds fleet rollout summary metrics and a live 15-second metric refresh.
- Renames the content workflow heading to Content Deployments.
- Adds a self-committing, self-tagging ZIP installer with automatic backups.

## Release tracking

The installer commits the release files to the local repository and creates the `v13.1.0` tag after validation succeeds.
