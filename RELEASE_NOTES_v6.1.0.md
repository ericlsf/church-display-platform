# v6.1.0 — Fleet Dashboard

## Landing page

The Hub now opens to `/fleet-dashboard` after login.

## Metrics

The dashboard shows:

- Displays online and offline
- Healthy, warning, and critical displays
- Updates available
- Active deployment jobs
- Failed jobs
- Sync issues

## Operational panels

- Needs Attention
- Active Deployments
- Recent Failures

Every affected display links directly to its individual details page.
