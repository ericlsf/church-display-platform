# v2.2.0 — Hub LAN Content Distribution

- Google Drive remains the authoritative content source.
- The Hub caches Drive media once and distributes it to displays over LAN.
- Displays use hash manifests and download only missing or changed files.
- Drive-direct synchronization remains available as an automatic fallback.
- Manual playlist order is stored on the Hub.
- New or modified Drive files are placed at the front automatically; existing manually ordered files retain their relative order.
